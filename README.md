# Recall-FSRS

A modern, open-source flashcard app for iOS built with SwiftUI.

Inspired by the power of Anki and the polish of modern Apple design — without having to choose between the two.

> ⚠️ This project is in early development. Things will change.

---

## What is this?

Recall-FSRS is a spaced repetition flashcard app that combines evidence-based scheduling (via the [FSRS algorithm](https://github.com/open-spaced-repetition/fsrs4anki)) with a clean, native iOS experience.

The goal is simple: make studying more effective and less annoying.

---

## Planned Features

- Flashcard creation and deck management
- FSRS spaced repetition scheduling
- Native SwiftUI interface
- iCloud sync across devices
- Import from Anki and Quizlet
- Free and open source, forever
